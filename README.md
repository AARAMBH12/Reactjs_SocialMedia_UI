
# React UI Project

A modern React-based user interface application that simulates a social media feed. This project showcases a responsive layout with posts, sidebar, rightbar, and a toggleable dark/light mode theme.

## Features

- **Social Media Feed**: Display a list of posts with user information, titles, descriptions, and images.
- **Interactive Elements**: Each post includes like, comment, and share counts with icons.
- **Sidebar and Rightbar**: Additional sections for navigation and featured content.
- **Dark/Light Mode Toggle**: Switch between dark and light themes dynamically.
- **Responsive Design**: Built with CSS for a clean and adaptable user interface.
- **React Icons**: Utilizes react-icons for consistent and scalable iconography.

## Tech Stack

- **Frontend Framework**: React 19.2.0
- **Build Tool**: Vite 7.3.1
- **Icons**: React Icons 5.5.0
- **Linting**: ESLint with React-specific rules
- **Styling**: Custom CSS

## Installation

1. Clone the repository:

   ```bash
   git clone <repository-url>
   cd reactsjsui
   ```

2. Install dependencies:

   ```bash
   npm install
   ```

3. Start the development server:

   ```bash
   npm run dev
   ```

4. Open your browser and navigate to `http://localhost:5173` (or the port shown in the terminal).

## Usage

- **Viewing Posts**: The main section displays a feed of posts fetched from static data.
- **Theme Toggle**: Use the toggle button in the navbar to switch between dark and light modes.
- **Navigation**: Explore the sidebar for additional options and the rightbar for featured content.

## Scripts

- `npm run dev`: Starts the development server with hot module replacement.
- `npm run build`: Builds the project for production.
- `npm run lint`: Runs ESLint to check for code quality issues.
- `npm run preview`: Previews the production build locally.
